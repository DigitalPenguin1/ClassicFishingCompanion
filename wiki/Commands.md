# Commands

All CFC slash commands start with `/cfc`.

---

## General

| Command | Description |
|---|---|
| `/cfc` | Open or close the main UI |
| `/cfc hud` | Toggle the Stats HUD (swaps gear if enabled) |
| `/cfc stats` | Print a stats summary to chat |
| `/cfc minimap` | Toggle the minimap button visibility |
| `/cfc debug` | Toggle debug mode (prints `[CFC Debug]` lines to chat) |

---

## Gear

| Command | Description |
|---|---|
| `/cfc savefishing` | Save your currently equipped gear as your fishing set |
| `/cfc savecombat` | Save your currently equipped gear as your combat set |
| `/cfc swap` | Swap between fishing and combat gear |

---

## Data

| Command | Description |
|---|---|
| `/cfc reset` | Reset all saved data (prompts for confirmation) |

---

## Debug Mode

`/cfc debug` toggles debug logging. When enabled, CFC prints detailed internal events to chat (lure detection, cast tracking, loot detection, etc.).

Turn debug off when you're done: `/cfc debug`
