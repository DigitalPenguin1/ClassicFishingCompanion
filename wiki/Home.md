# Welcome to Classic Fishing Companion

**Classic Fishing Companion (CFC)** is a comprehensive fishing addon for **World of Warcraft Classic Era** and **TBC Classic**. It tracks your catches, manages your gear, auto-applies lures, and helps you fish smarter â€” all from a clean in-game UI.

---

## Quick Start

1. Install the addon and log into WoW
2. Click the **minimap button** or type `/cfc` to open the UI
3. Start fishing â€” all catches are tracked automatically

> **Tip:** Right-click the minimap button to toggle the HUD. Hover over it for a quick stats summary.

---

## Features at a Glance

| Feature | Description |
|---|---|
| **Stats HUD** | Draggable overlay showing session catches, fish/hour, skill, and lure timer |
| **Easy Cast** | Double right-click to cast; auto-applies your selected lure when it expires |
| **Gear Sets** | Save fishing and combat gear sets; auto-swaps when combat starts |
| **Catch List** | Full catch history with fish/misc separation, icons, and zone breakdown |
| **Zones Tab** | Collapsible zone view showing every fish caught per zone |
| **Statistics** | Top catches, session vs. total, fish per hour, and more |
| **Lure Manager** | Select and apply lures directly from the HUD |
| **Goals** | Set session-based catch goals with HUD progress tracking |
| **Catch & Release** | Mark fish to auto-delete via keybind â€” catches still count |
| **Alerts** | Rare fish sounds, milestone notifications, missing lure warnings |
| **Keybindings** | Custom keybinds via ESC > Settings > Key Bindings |
| **Data Backup** | Auto-backup every 24 hours with one-click restore |
