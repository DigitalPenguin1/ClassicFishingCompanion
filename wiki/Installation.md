# Installation

## CurseForge (Recommended)

1. Install the [CurseForge App](https://www.curseforge.com/download/app)
2. Search for **Classic Fishing Companion**
3. Click **Install** — the app handles everything automatically
4. Launch WoW and enable the addon in the **AddOns** menu on the character select screen

---

## Manual Install

1. Download the latest release from [CurseForge](https://www.curseforge.com/wow/addons/classic-fishing-companion) or [GitHub Releases](https://github.com/DigitalPenguin1/ClassicFishingCompanion/releases)
2. Extract the zip file
3. Copy the `ClassicFishingCompanion` folder into your WoW addons directory:

**Classic Era:**
```
World of Warcraft\_classic_era_\Interface\AddOns\
```

**TBC Classic:**
```
World of Warcraft\_classic_\Interface\AddOns\
```

4. Launch WoW and enable the addon in the **AddOns** menu on the character select screen

---

## Enabling the Addon

- On the character select screen, click **AddOns** (bottom-left)
- Find **Classic Fishing Companion** in the list and check the box
- Click **Okay** and log in

---

## First Launch

Once in-game, you'll see:

- A **minimap button** (fishing bobber icon) — left-click to open the UI, right-click to toggle the HUD
- A **What's New** popup showing the latest features
- Type `/cfc` in chat at any time to open the UI

---

## Transferring Data from Classic Era to TBC

1. **Exit WoW completely** (Exit Game, not just log out)
2. Copy both files from `_classic_era_` to the matching paths in `_classic_` (TBC):
   - `WTF\Account\[ACCOUNT]\SavedVariables\ClassicFishingCompanion.lua`
   - `WTF\Account\[ACCOUNT]\[SERVER]\[CHARACTER]\SavedVariables\ClassicFishingCompanion.lua`
3. Launch TBC and log in — your data will carry over
